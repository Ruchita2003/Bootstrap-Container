Que-4 What is the difference between Bootstrap 4 and Bootstrap 5?

Ans- here are some key differences between Bootstrap 4 and Bootstrap 5: Default CSS/JS Split: Bootstrap 5 separates CSS and JavaScript by default, making it more modular and improving load times compared to Bootstrap 4. Vanilla JavaScript: Bootstrap 5 introduced a shift towards using vanilla JavaScript instead of jQuery for its components, reducing overall dependencies. New Components and Utilities: Bootstrap 5 added new components like the accordion, off-canvas, and improved utilities for spacing and sizing, enhancing customization options for developers.

Que-7 In Bootstrap 4, what is flexbox?

Ans-In Bootstrap 4, flexbox is a layout model that provides an efficient and flexible way to structure and align elements within a container. It is a modern CSS layout technique that allows for the creation of complex and responsive layouts with ease. The flexbox model in Bootstrap 4 enables developers to create a variety of dynamic and responsive designs, making it simpler to align, distribute, and reorder elements within a container.
